hyperhq/jenkins-slave-android:android-21
========================================

This is a Jenkins slave image based on `ubuntu:14.04`.

It contains:
- oracle-java8
- `android sdk:r24.4.1`
- `gradle:3.1`
- jenkins slave(REF: [jenkinsci/slave](https://hub.docker.com/r/jenkinsci/slave/))

Usage: `./build.sh`
